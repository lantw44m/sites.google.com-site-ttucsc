夏日的 Web 樂園 2012
posted by Denny Huang on 七月 13, 2012

詳細資訊請見活動網頁: http://moztw.org/events/scp0712/


Mozilla Webmaker - 夏日的Web樂園

網際網路發展至今，已經從第一階段的上網閱讀網頁資訊，第二階段的使用網路服務，而開始邁向第三個階段：讓所有人都開始在網路上創作。就像每人都能提筆寫字一樣、人人都應該能夠在網路上留下自己的足跡。Mozilla 的 Webmaker 計畫即是以簡便的工具，讓人們學習撰寫簡單網頁的技能。


本次「夏日的Web樂園」由海洋大學網路發展協會 、MozTW 社群及美商謀智（Mozilla Taiwan），邀請各社群共同籌辦，以園遊會呈現各種參與網路的方式，帶領你在網路上留下足跡！只要你平常喜歡上網，想多瞭解網頁的原理，都歡迎加入！

參與團隊（按字母序）

  • 大同大學 資訊創意研究社
  • 國立台灣海洋大學 網路發展協會
  • Mozilla Taiwan 美商謀智
  • MozTW (Mozilla 台灣社群)
  • 台灣數位文化協會 胖卡小組
  • Wordpress Taiwan

時間地點

2012 年 7 月 29 下午 2:00 至 下午 4:30
地點：華山1914文創園區 中3拱聽

活動形式

活動的形式為 Mozilla Summer of Code 的「Pop Up」，類似園遊會形式，由各單位設置一或多個動手做的攤位，帶領使用者以不同方式，透過網路服務進行創作，在網路上留下足跡。每一個攤位都有不同的主 題，例如幫網路影片製作字幕、透過簡單機制混搭影片、依照樣板做張網路卡片……等。參加者可以自由參與各主題，或者與他人交流。
