還宅在家裡寫程式？試試自由軟體開發模式

上課時間 :  5 / 22 (二)
還宅在家裡寫程式？試試自由軟體開發模式：   下午 18：00 - 20 : 30

上課地點 : 尚志大樓 814 電腦教室

講者 :    王家薰
服務於中央研究院資訊科學所自由軟體鑄造場。熱愛自由軟體的小工程師，喜歡研究並利用OpenSource 的工具打造親合力實足的網站，目標打造一套能夠帶給人們快樂及綠色地球的系統。(握)．
      • SKILL：PHP、Linux、Joomla、BSD、CSS、HTML、JavaScript、LAMP、UI

主題領域 : 軟體開發

所需具備能力 : Linux操作基礎

課程簡介 :

身為軟體工程師的了解 Open Source 對於你未來的幫助將超出你想像，本課程將深入淺出的引領聽眾了解自由軟體的開發模式。

  • Open Source 和你的關係有多密切。
  • Open Source 可以幫助我們什麼。
  • Open Source 開發模式
