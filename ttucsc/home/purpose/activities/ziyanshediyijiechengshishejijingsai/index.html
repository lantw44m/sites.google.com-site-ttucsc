資研社第一屆程式設計競賽

時間&地點
100/03/31(四) 17:20~20:30 尚志大樓 8樓 814 電腦教室

活動對象:
大同大學 大學部 大一~大三 學生

計分方式:
積分制，根據題目難度、年級及系別不同，給予不同配分，以利公平競爭。
注意:比賽期間禁止使用任何通訊設備且不可使用任何私人儲存設備，並會切斷網路連線，”可”攜帶任何紙本參考書籍或文件，數量不限

獎金:
第一名 新台幣 1500 元整 及 獎狀乙紙
第二名 新台幣 1000 元整 及 獎狀乙紙
第三名 新台幣  500 元整 及 獎狀乙紙

競賽時程:
17:20~17:40 報到 & 環境測試 (逾時視同放棄參賽資格)
17:40~18:00 比賽規則說明 & 設定上傳位置
18:00~20:00 競賽解題時間
20:00~20:30 成績統計 & 頒獎
各題之解題技巧及方法將於隔週同一時間社課講解

報名方式:
一律採取網路報名至3/28截止 (網址請注意大小寫)
  1. 至社團網站( http://goo.gl/X086X ) 報名頁面報名
  2. 至http://goo.gl/3JAL0 填寫報名表單

814電腦教室電腦數量有限，若超過可負荷人數，則依報名時間順序保留三名候補參賽者
報到時須出示學生證以供核對身分

報名費用:
比賽使用之試題多數為評審原創出題，並提供題本供參賽者計算筆記之用，顧慮其成本及獎金，本社社員可免費參與，非本社社員需於報到時繳交100元報名費，或可當日繳交200元社費以社員資格參與

使用語言:
C, C++

提供IDE:
Dev C++, Visual Studio 6.0, Visual Studio 2005, Visual Studio 2008
